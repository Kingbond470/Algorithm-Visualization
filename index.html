Algorithm Visualizer

Algorithm Visualizer is an interactive way and my web platform that visualizes the algorithms in two domains i.e. Path Finding Algorithm and Sorting Visualizer. The project focuses on analyzing the problem faced during the understanding and learning the algorithms.

Algorithm Visualizer is an interactive way and my web platform that visualizes the algorithms in two domains i.e. Path Finding Algorithm and Sorting Visualizer. The project focuses on analyzing the problem faced during the understanding and learning the algorithms.

After analyzing the current process of learning and understanding the algorithm, I came up with the idea of visualization.

Features

Path Finding

At its core, a pathfinding algorithm seeks to find the shortest path between two points.Some of the algorithms are BFS, DFS, Dijkstra’s and A* algorithm. Additionally, not all algorithms guarantee the shortest path.Some of the algorithms are BFS, DFS, Dijkstra’s and A* algorithm. Additionally, not all algorithms guarantee the shortest path.

Sort Algorithm

Array size and representation. Sorting the array of items with required speed and acquire patterns.Set of algorithm which will be used for sorting is Bubble sort, Merge sort, Quick sort, Heap sort, etc. Enjoy learning with your own speed i.e. slow, medium, fast!

Other Feature

Adding walls and weight. Generate mazes and patterns from the "Mazes & Patterns". Walls are impenetrable,Adding a bomb. It will change the course of the chosen algorithm. In other words, the algorithm will first look for the bomb and will then look for the target node. Dragging nodes. You can clear the current path, clear walls and weights, clear the entire board, and adjust the visualization speed.
